Advanced React E-Commerce Web App
---
Hello, this is an updated fake e-commerce web application I made for Coding Temple.

The application serves as a mock storefront for browsing, selecting, and purchasing different items called via the fakestore API for demonstration purposes. The application is coded in React and styled with Bootstrap.

The e-commerce application contains the following features:

* Product Listings and Display with Info

* Category Navigation

* Shopping Cart with a Checkout Functionality
