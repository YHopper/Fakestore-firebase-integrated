Mock React E-Commerce Web App with Firebase Integration
---
Hello, this is an updated fake e-commerce web application I made for Coding Temple.

The application serves as a mock storefront for browsing, selecting, and purchasing different items called via a collection from Firebase for demonstration purposes. In addition, this application is made to have user management and authentication, product management, and order managment.  The application is coded in React and styled with Bootstrap.

The e-commerce application contains the following features:

* User Authentication and Managment
  
* Product Listings and Display with Info

* Product Management

* Category Navigation

* Shopping Cart with a Checkout Functionality

* Order Management
